class is a collection of data members and members functions.
public , private are access specifiers . public can be accessed from outside a class . private can be accessed within class.
statci access modifer. members declared as statci can be accessed using the class name of the class. no need to create the object.
void means no return value.
main() entery point program execution starts from main.
